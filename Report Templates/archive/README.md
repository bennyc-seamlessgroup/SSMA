# Archived Report Material

These files are retained for visual reference and historical comparison. They are not loaded by the portal runtime.

- `comprehensive-daily-close-v2/` - archived comprehensive report design
- `old-files/` - superseded report implementation files
- `sample-reports/` - generated report examples and rendered previews
- `sample-report-generators/` - scripts used to create the older sample reports

The current maintainable report source remains in `Report Templates/currenc-closing-digest-report-demo/`.
