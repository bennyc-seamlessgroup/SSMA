# Report Templates

## Active report

The current lean daily market-close report is maintained in:

```text
lean-daily-market-close-report/
```

That folder contains the HTML template, CSS, renderer, sample payload, backend data contract, PDF generator, and generated visual preview.

The Report Archive page uses the matching browser runtime files in:

```text
public/report-templates/daily-close/
```

## Archived reports

Superseded templates, generated samples, and previous comprehensive report designs are retained under:

```text
archive/
```
